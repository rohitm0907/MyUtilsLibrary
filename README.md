# MyUtilsLibrary
Shortcut functions to use in your project


Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
Add the dependency

	dependencies {
	        implementation 'com.github.rohitm0907:MyUtilsLibrary:1.0.0'
	}
